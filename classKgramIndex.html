Detailed Description

The KgramIndex class takes in an integer n, and that is the number kgram's number of char grams. Main needs to send in an int n and that is what the kgram separates to.

Index for Kgrams to map from kgram to list of terms; has add and get (public) functionality for the index; essentially Index object, and a few independent methods that allow you to use kgram logic.

addTerm adds the term to the KgramIndex. Breaks the term into kgrams of ksize and adds into the index.

Parameters
termstd::string of term that you wish to add to the index.

This (public) method will separate the term into kgrams (of the size passed in constructor) and use the addKgram (private) to add that kgram key to term pair into the _mIndex.
